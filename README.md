# 🔨 裝潢報價系統

> 輕量級、免安裝的裝潢工程報價單產生器，一個 HTML 即可運行。

🔗 **線上使用**：[https://dada3c.github.io/quote-system/price.html](https://dada3c.github.io/quote-system/price.html)

---

## ✨ 功能特色

| 功能 | 說明 |
|------|------|
| 📝 **即時報價計算** | 新增項目、數量、單價，自動即時計算小計與總計 |
| 💾 **自動暫存** | 所有資料自動存入 localStorage，關閉重開不遺失 |
| 📊 **匯出 Excel** | 一鍵匯出 `.xlsx` 格式，含完整欄寬設定 |
| 📄 **匯出 PDF** | 生成 A4 格式報價單 PDF，可直接傳給客戶 |
| 🧮 **折扣與稅金** | 支援百分比折扣、固定折讓、5% 營業稅、自訂稅率 |
| 🏢 **公司資訊設定** | 自訂公司名稱、電話、地址、報價有效天數 |
| 📋 **項目範本庫** | 內建 6 大工程分類、20+ 常用項目快速選取 |
| ↕️ **拖曳排序** | 支援拖曳或上下箭頭調整項目順序 |
| ↩️ **刪除復原** | 誤刪項目後 5 秒內可一鍵復原 |
| 📱 **行動裝置優化** | RWD 設計，手機平板皆可操作 |

---

## 🚀 使用方式

### 線上使用
直接開啟 [GitHub Pages](https://dada3c.github.io/quote-system/price.html)，無需安裝。

### 本機使用
下載 `price.html`，用瀏覽器直接開啟即可（需要網路載入 CDN 資源）。

## 🛠️ 技術架構

- **純前端**：單一 HTML 檔案，零後端依賴
- **TailwindCSS** (CDN)：UI 樣式
- **FontAwesome**：圖示
- **SheetJS (xlsx)**：Excel 匯出
- **html2pdf.js**：PDF 生成
- **localStorage**：資料暫存
